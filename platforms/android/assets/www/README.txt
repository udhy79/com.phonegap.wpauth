# Wordpress

In order to use this project you should have installed in your Wordpress installation:

* JSON API - A RESTful API for WordPress
https://wordpress.org/plugins/json-api/

* JSON API Auth - Extends the JSON API Plugin for RESTful user authentiocation
https://wordpress.org/plugins/json-api-auth/

* JSON API User - Extends the JSON API for RESTful user registration, authentication, password reset, Facebook Login, user meta and BuddyPress
https://wordpress.org/plugins/json-api-user/


# Wordpress Authentication App
 
* Edit your www/js/app.js file and enter your API BASE URL (api_url) that you have defined to JSON API Settings

* Edit your www/js/data.js file and enter your Server Posts Data URL
